# 🎓 Student Performance Dashboard | Power BI

## 📌 Project Overview

This project presents an interactive Power BI dashboard designed to analyze student academic performance and identify trends across subjects, attendance, and overall achievements.

The dashboard enables educators and stakeholders to monitor key performance indicators (KPIs), compare subject-wise scores, and gain actionable insights to improve academic outcomes.

---

## 🎯 Business Problem

Educational institutions need an efficient way to monitor student performance and identify trends that affect academic success.

This dashboard helps answer questions such as:

- Which students are performing well?
- Which subjects have lower average scores?
- How does attendance impact performance?
- What are the overall academic trends?

---

## 🎯 Objectives

- Analyze student academic performance
- Compare subject-wise scores
- Track attendance trends
- Monitor KPIs related to student achievements
- Provide interactive visualizations for better decision-making

---

## 📂 Dataset Information

The dataset contains information about:

- Student ID
- Student Name
- Gender
- Grade/Class
- Subject
- Marks/Score
- Attendance Percentage
- Pass/Fail Status

---

## 🧹 Data Preparation

The dataset was prepared using Power Query with the following steps:

- Checked missing values
- Removed duplicate records
- Verified data types
- Renamed columns for consistency
- Created calculated columns where required
